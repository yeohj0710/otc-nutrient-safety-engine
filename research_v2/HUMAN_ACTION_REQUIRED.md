# Human actions required

통합 입력 양식: `권혁찬_연구_사람작업패킷_20260710.xlsx`. 파일명과 SHA-256은 `audit/human_action_packet_manifest.json`에 기록했다.

## H-001 — 프로토콜과 K1-K5 승인

- 필요한 사람: 지도교수 또는 지정 연구 검토자
- 필요한 입력: 서명본, 승인 이메일, 또는 날짜·참석자·승인 문구가 있는 회의록
- 승인 내용: K1-K5, 포함·제외 기준, 핵심 결과, 집중 합성 선정 규칙, AI 평가와 독립 검증 계획
- 자동화 불가 이유: 연구 범위와 임상적 중요도에 대한 책임 있는 승인임
- 차단 작업: Gate 1 동결, 등록, 최종 검색식 실행
- 계속 가능한 작업: 검색식 초안 검증, seed 세트 준비, 코드·스키마·환경 테스트
- 상태: open

## H-002 — 검색전략 동료 검토

- 필요한 사람: 정보전문가, PRESS 경험 검토자, 또는 지도교수가 지정한 동료 검토자
- 필요한 입력: `research_v2/search/peer_review.csv`의 항목별 판정·수정 사유·reviewer ID·날짜
- 자동화 불가 이유: 통제어, 근접연산자, 누락 위험의 독립 검토가 필요함
- 차단 작업: Gate 2 최종 검색식 승인
- 상태: open

## H-003 — 접근 제한 데이터베이스 전체 export

- 필요한 사람: 연세대학교 기관 인증 사용 권한이 있는 연구자
- 필요한 입력: Embase, CENTRAL, Scopus 또는 Web of Science의 전체 RIS/CSV export와 검색 화면 hit 수·플랫폼·검색일
- 자동화 불가 이유: 기관 라이선스와 사용자 인증이 필요함
- 차단 작업: Gate 2 완료와 최종 검색 분모 확정
- 상태: open

## H-004 — 제2검토자와 조정자 판정

- 필요한 사람: 제2검토자 1명과 불일치 조정자 1명
- 필요한 입력: reviewer ID·UTC timestamp·독립 판정·조정 결과가 있는 CSV
- 자동화 불가 이유: AI 출력은 최종 선별·추출·RoB·GRADE 판정자가 될 수 없음
- 차단 작업: Gate 3-5, evidence freeze
- 상태: open

## H-005 — 독립 시나리오와 전문가 내용타당도

- 필요한 사람: 최소 2명의 약사 또는 임상 검토자와 조정자
- 필요한 입력: 개발 규칙을 보지 않고 작성한 scenario gold, 독립 판정, I-CVI/S-CVI 양식
- 자동화 불가 이유: 임상 위해·행동 등급의 골드 판정은 전문가 책임임
- 차단 작업: Gate 8과 연구 릴리스
- 상태: open

## H-006 — IRB 또는 면제 판단

- 필요한 사람: 기관 연구윤리 담당부서
- 필요한 입력: 승인번호, 면제 확인서, 또는 사람 대상 모집을 하지 않는다는 공식 범위 결정
- 자동화 불가 이유: 기관 규정에 따른 법적·윤리적 판단임
- 차단 작업: 실제 사용자 모집·인터뷰·사용성 시험만 차단
- 상태: open

## H-007 — NCBI E-utilities 연락 이메일

- 필요한 사람: 연구 책임자 또는 학생 연구자
- 필요한 입력: NCBI E-utilities 요청에 사용할 유효한 연락 이메일을 로컬 `NCBI_EMAIL` 환경변수로 설정
- 저장 금지: 이메일 값은 git과 공개 산출물에 기록하지 않음
- 자동화 불가 이유: 연락 가능한 실제 사용자의 주소여야 함
- 차단 작업: PubMed API 최종 검색 실행
- 계속 가능한 작업: 검색식·seed·pipeline 검증
- 상태: open

## H-008 — 공식 졸업논문 서식 확인

- 필요한 사람: 연세대학교 약학대학 행정 담당자 또는 지도교수
- 필요한 입력: 최신 공식 DOCX/HWP 서식, 또는 기존 권혁찬 DOCX의 제출 서식 사용 승인
- 자동화 불가 이유: 현재 확인된 DOCX가 학교 공식 최신 서식이라는 증거가 없음
- 차단 작업: Gate 9 최종 DOCX/PDF 확정
- 계속 가능한 작업: 기존 DOCX의 레이아웃 구조 감사, 논문 생성기와 렌더 검사 준비
- 상태: open
