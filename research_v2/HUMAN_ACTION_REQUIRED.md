# 사람 확인이 필요한 잔여 작업

PubMed 단독 타당성 연구는 완료했다. 아래 항목은 현재 연구본을 다중 데이터베이스 체계적 문헌고찰 또는 임상 검증 도구로 확장할 때만 필요하다. 미확보 상태를 결과값 0 또는 합격으로 대체하지 않는다.

## 현재 완료된 범위

- 권혁찬(2021194024) 연구 정체성 및 Gate 0 확인
- PubMed K1–K5 전량 검색, UID·XML·SHA-256 보존
- 사전 seed 22/22 회수
- 16,194건 occurrence, 15,890건 고유 레코드 생성
- 계산 선별 제안 15,890건 및 초록 후보 118건 생성
- 사전 seed 22건의 PubMed 초록 locator 확인
- `draft_ai` 규칙 5개 생성, released 규칙 0개 유지
- 연구본 DOCX/PDF 생성 및 7쪽 렌더링 검사

## H-001 프로토콜·범위 승인

- 필요한 사람: 지도교수
- 입력: amendment-001 및 K1–K5 범위 승인 기록
- 미확보 영향: 본 연구본은 학생·에이전트 수행 타당성 연구이며 승인된 최종 체계적 문헌고찰로 주장할 수 없음
- 상태: open, 현재 축소 범위의 기술 실행은 완료

## H-002 검색전략 독립 검토

- 필요한 사람: PRESS 경험 정보전문가 또는 독립 문헌검색 검토자
- 입력: `search/peer_review.csv`의 항목별 검토와 서명
- 미확보 영향: seed 22/22 회수는 확인했으나 독립 검토 검색식으로 부를 수 없음
- 상태: open

## H-003 기관 데이터베이스 export

- 필요한 사람: Embase, CENTRAL, Scopus 또는 Web of Science 접근 권한 보유자
- 입력: 전체 RIS/CSV export와 검색일·hit 수
- 미확보 영향: PubMed 밖의 근거 누락을 평가할 수 없음
- 상태: out_of_scope_for_feasibility, expansion 시 open

## H-004 이중 선별·전문 추출·RoB·GRADE

- 필요한 사람: 독립 검토자 2명과 불일치 조정자
- 입력: 최종 선별, 전문, 독립 추출, RoB, GRADE 판정 CSV
- 미확보 영향: 포함 연구 확정, 효과크기 합성, 근거 확실성 판정, 규칙 release 금지
- 상태: open

## H-005 독립 시나리오·전문가 내용타당도

- 필요한 사람: 약사 또는 임상 검토자 최소 2명
- 입력: 개발 규칙을 보지 않고 만든 독립 scenario gold와 I-CVI/S-CVI
- 미확보 영향: 위험 탐지 민감도, critical false negative, 전문가 타당도는 `not_evaluated`
- 상태: open

## H-006 IRB 또는 면제 판단

- 필요한 사람: 기관 연구윤리 담당부서
- 입력: 승인번호, 면제 확인 또는 사람 대상 자료를 사용하지 않는다는 공식 판단
- 미확보 영향: 사람 모집·인터뷰·사용성 시험 금지
- 상태: open

## H-007 NCBI 연락처

- 실제 검색은 유효한 로컬 `git config user.email`을 NCBI 요청 연락처 fallback으로 사용했다.
- 이메일 값은 산출물에 저장하지 않았다.
- 상태: resolved_for_local_execution

## H-008 학교 공식 서식 확인

- 필요한 사람: 연세대학교 약학대학 행정 담당자 또는 지도교수
- 입력: 최신 공식 DOCX/HWP 서식 승인
- 미확보 영향: 현재 DOCX/PDF는 참고문서에서 추출한 A4 형식의 연구본이며 공식 제출본 확정은 아님
- 상태: open
