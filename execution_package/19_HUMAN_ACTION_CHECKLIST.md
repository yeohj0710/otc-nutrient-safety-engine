# 사람이 반드시 확인해야 하는 항목

Codex는 아래 항목을 사실처럼 꾸며 완료할 수 없다. 필요한 문서와 양식을 준비하고, 미응답 상태를 명시하면서 나머지 작업을 진행한다.

1. 지도교수의 최종 연구 유형·제목·5개 노드 승인
2. Embase, CENTRAL, Scopus/Web of Science의 기관 접근
3. 검색전략 동료/정보전문가 검토
4. 두 번째 문헌 선별·원문 제외 검토자
5. RoB·GRADE의 분쟁 조정자
6. 임상 시나리오 골드 라벨을 만드는 약사/임상 전문가
7. 규칙 내용 타당도 패널
8. 사용자 연구를 수행할 경우 IRB 또는 면제 확인
9. 학교 최종 제출 서식과 제출일
10. 최종 논문 저자 확인과 AI 사용 공개 문구

각 요청은 `HUMAN_ACTION_REQUIRED.md`에 다음 형식으로 기록한다.

```text
ID:
필요한 사람/권한:
구체적 요청:
제공 형식:
마감일:
이유:
막히는 작업:
막히지 않는 작업:
상태:
```
