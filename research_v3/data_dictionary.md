# research_v3 데이터 사전

활성 데이터 사전은 `research_v3/otc/data_dictionary.md`다. 국내 일반의약품 제품, 성분, 제품-성분-규격, 분석 제외, 제품별 복용 조건, 규칙, 평가와 런타임 감사 필드를 정의한다.

기존 `screening`, `extraction`, `search`와 영양성분 중심 규칙 자료는 superseded 계보다. 기존 파일은 감사 재현을 위해 보존하지만 활성 OTC 결과에 합산하지 않는다.

## 활성 수치의 데이터 층

- 후보·허가 원문 마스터: 제품 16개, 성분 31개, 제품-성분-규격 106행
- 분석 제품의 전체 규격: 제품 13개, 성분 28개, 57행
- 사이트 계산 런타임: 제품 13개, 성분 28개, 제품-성분 연결 47개
- 허가 원문 검증 복용 조건: 32개

논문과 사이트의 주 분석 결과는 사이트 계산 런타임 층을 사용한다. 후보 마스터 수치는 원자료 보존 범위를 설명할 때만 사용한다.
