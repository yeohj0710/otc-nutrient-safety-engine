# OTC 외부 검토 안내

## 최종결정권자 확인

`OTC_최종결정권자_확인.html`에는 Codex가 규칙 16건과 시나리오 13건의 제안 결정을 모두 작성해 두었다. 최종결정권자는 목록을 확인하고 맨 아래 `전체 최종안 확인` 버튼을 한 번만 누른다. 이 결과는 비맹검 최종결정 확인으로 기록되며 약사 전문가 검토 또는 맹검 독립평가를 대신하지 않는다.

## 역할별 전문 검토

1. 약사·약학 전문가는 `OTC_약사전문가_검토.html`, 독립 시나리오 검토자는 `OTC_독립시나리오_검토.html`을 Chrome 또는 Edge로 연다.
2. 검토자 코드와 역할은 미리 고정되어 있다. 실제 검토자가 해당 역할 자격과 내용을 직접 확인한다.
3. 화면에 제시된 제품·근거·규칙 또는 독립 시나리오를 확인한다.
4. 약사·약학 전문가는 추천안을 확인하며 `추천안 승인하고 다음`만 누른다. 독립 시나리오 검토자는 맹검 유지를 위해 각 사례의 `위험 신호 있음/없음/판단 보류`를 직접 선택한 뒤 다음을 누른다.
5. 저장한 JSON을 연구 담당자에게 전달한다.

외부 검토자는 본인이 실제로 확인한 역할만 선택해야 한다. 시스템이 미리 작성한 후보 판정은 사람 검토 결과가 아니다. 독립 시나리오 검토자는 시스템 예측을 보지 않은 상태에서 기준정답을 확정해야 한다.

현재 규칙은 모두 초안이며 released 규칙은 0건이다. 약사·약학 전문가 승인 후에도 근거 locator와 runtime binding을 별도로 점검한 뒤에만 승격할 수 있다. 정량 근거가 없는 장기복용 제한 규칙은 자동 승격되지 않는다.

프리뷰는 비공개 보호 URL이며, 공개 운영 배포가 아니다.
