# 국내 일반의약품 안전성 연구 작업본

이 폴더는 최종 승격 전 작업본과 외부 검토 자료를 모은 패키지다. 기존 `권혁찬_졸업논문_최종본`을 덮어쓰지 않았다.

- `01_논문_작업본`: OTC 방향으로 다시 작성한 논문 DOCX/PDF/원문
- `02_연구계획서_작업본`: OTC 연구계획서 DOCX/PDF/원문
- `03_보고서_및_검증`: 연구 보고서, 지표, 완료 경계와 검증 결과
- `04_외부검토`: 외부 검토자가 열어 볼 통합 HTML, 역할 안내, 독립 시나리오 원본
- `05_프리뷰검증`: 비공개 프리뷰 검증 기록과 데스크톱·모바일 캡처

외부 검토자는 `04_외부검토/외부검토_안내.md`를 읽고 본인 역할에 맞는 `OTC_약사전문가_검토.html` 또는 `OTC_독립시나리오_검토.html`을 열면 된다. 본인의 ID로 검토한 뒤 JSON을 저장한다.

현재 상태는 `release_ready=false`다. released 규칙 0건이며, 독립 평가도 완료되지 않았다. 공개 운영 배포와 canonical 최종본 승격은 승인 범위에서 제외되었다.
