# 국내 일반의약품 안전성 조회 연구 상태 보고서

기준일: 2026-07-15  
연구자: 권혁찬(2021194024), 연세대학교 약학대학

## 현재 결론

활성 연구는 국내 실제 일반의약품을 제품명으로 선택하고, 검증된 성분·함량·복용 조건을 불러와 중복 성분, 동일 약리군, 용량·간격, 연령·질환·병용약 위험 신호와 근거를 확인하는 시스템의 개발과 평가다. 기존 영양성분 연구는 superseded 계보로만 보존한다.

후보·허가 원문 마스터는 16개 제품, 31개 고유 성분과 106개 제품-성분-규격 행이다. 주 분석·사이트 집합은 13개 제품, 28개 고유 성분과 47개 계산용 제품-성분 연결이다. 이 두 데이터 층을 같은 의미의 제품-성분 연결 수로 사용하지 않는다.

신신파스아렉스는 4매 포장이 7×10㎠와 10×14㎠에 모두 허가되어 규격을 하나로 결정할 수 없다. 원시 HTML·PDF, 제품·성분 마스터와 49개 규격 행은 보존하되 분석, 사이트 검색·계산 대상과 런타임 미지원 후보에서 제외하였다.

제품별 복용 조건 32개는 1회 최대 단위 12개, 하루 최대 횟수 13개, 성분별 최대 1일량 5개와 최소 간격 2개다. 각 행은 허가 용법·용량 PDF의 source, locator와 SHA-256에 연결된다. 이 조건은 2026-07-14 외부 약사 규칙 검토 이후 추가되었으므로 약사 검토 완료라고 표시하지 않는다.

## 사이트와 연구자료 정합성

`runtime_research_alignment.json`은 분석 CSV와 사이트 런타임 JSON을 제품 ID, 성분 ID, 단위당 함량, 단위, source, locator와 복용 조건 ID 수준에서 비교한다. 감사 결과는 `valid=true`다.

- 분석 제품 13개 = 런타임 제품 13개
- 분석 고유 성분 28개 = 런타임 고유 성분 28개
- 계산 연결 47개 = 런타임 성분 연결 47개
- 허가 복용 조건 32개 = 런타임 복용 조건 32개
- 액상제 단위 환산 오류 0건
- 분석 제외 제품 런타임 유입 0건

## 평가 상태

외부 확인 시나리오는 13건이다. 위험 11건과 비위험 2건에서 엔진 판정과 확인 결과가 모두 일치했고 FP 0, FN 0, critical FN 0이었다. 민감도, 특이도와 정확도의 계산값은 1.00이다.

그러나 평가자가 Codex 예상 답안을 볼 수 있었다. 검토 방식은 `codex_prefilled_external_human_confirmation`이고 `independent_blinding=false`다. 이 수치를 블라인드 독립 성능으로 주장하지 않으며 `performance_claim_allowed=false`를 유지한다.

## 문서 상태

논문은 `국문초록 → Abstract → 서론 → 연구방법 → 연구결과 → 고찰 → 결론 → 참고문헌 → 부록` 순서로 재구성하였다. 논문과 계획서는 후보 마스터 수치와 분석·사이트 수치를 분리하고, 신신파스아렉스 제외, 복용 조건 32개와 비블라인드 평가 한계를 반영한다.

## 소프트웨어 검증

최신 검증에서 연구 테스트 106개와 앱 테스트 53개가 통과했다. lint, typecheck와 production build도 통과했으며 정적 경로 156개를 생성했다. 이 결과는 데이터 계약과 구현 재현성을 확인한 것이며 임상 성능 증거가 아니다.

## 완료 조건

완료 감사 16개 조건 중 15개를 충족하였다. 유일한 미완료 조건은 `independent_evaluation_complete`다.

- `complete=false`
- `release_ready=false`
- `performance_claim_allowed=false`
- `independent_evaluation_complete=incomplete`

블라인드 독립평가가 실제로 완료되기 전에는 연구 전체 완료, 임상 성능 또는 안전 보장을 주장하지 않는다.

## 다음 사람 작업

규칙과 예상 답안을 보지 않은 독립 평가자가 무라벨 사례를 판정해야 한다. 제품별 복용 조건 32개에 대한 약사 재검토도 권장하지만, 현재 완료 감사의 유일한 필수 미완료 조건은 블라인드 독립평가다.
