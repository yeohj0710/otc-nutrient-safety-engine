# 2026-07-10 PubMed 검색 상태

## 판정

`provisional_reusable_search`, 최종 검색 아님.

## 확인된 사실

- K1-K5 전체 검색 결과 16,194건을 저장했다.
- PMID 기준 고유 문헌은 15,890건이다.
- 중복 후보 304쌍이 기록됐다.
- 현재 검색 로그가 참조한 5개 실행의 export/import 건수가 일치한다.
- 원시 XML·JSON 109개 파일의 크기와 SHA-256가 manifest와 모두 일치한다.
- 관련도 상위 N개만 저장한 검색이 아니다.

## 최종 검색으로 부를 수 없는 이유

- PRESS 기반 검색전략 동료 검토 패킷은 K1-K5 각 7항목, 총 35행으로 준비됐지만 완료 판정은 0건이다.
- Embase, CENTRAL, 인용색인 접근·실행이 완료되지 않았다.
- dedup 304쌍은 결정론적 완전 일치 후보이며 사람 확인이 없다.
- 정규화 행의 `raw_source_sha256`가 비어 있다. 원시 파일 단위 manifest는 유효하지만 행별 연결은 보완해야 한다.
- 제목·초록 및 전문 사람 판정이 0건이다.

## 사용 경계

검색 코드, 질의 초안, 원시자료, 정규화 구조, 재현 로그는 재사용할 수 있다. 문헌 포함 결론, 안전성 결론, released 규칙, 성능 지표에는 직접 사용하지 않는다.
