# 사람이 해야 하는 남은 작업

## 1. 블라인드 독립평가

연구 완료를 위해 반드시 필요한 유일한 항목이다. 현재 13개 외부 확인 결과는 평가자가 Codex 예상 답안을 본 상태에서 확정되었다. 엔진 판정과 13건 모두 일치했지만 블라인드 독립평가가 아니므로 성능 근거로 사용할 수 없다.

새 평가자는 다음 조건을 충족해야 한다.

1. 엔진 예측, Codex 예상 답안과 기존 사람 라벨을 보지 않는다.
2. `research_v3/otc/validation/independent_cases`의 무라벨 사례만 본다.
3. 위험 신호 유무와 critical 여부를 독립적으로 확정한다.
4. 검토자 ID, 검토 시각과 판단 근거를 기록한다.
5. 라벨을 잠근 뒤에만 엔진 예측을 연결하고 지표를 계산한다.

이 절차가 끝날 때까지 다음 상태를 바꾸지 않는다.

- `independent_blinding=false`
- `performance_claim_allowed=false`
- `independent_evaluation_complete=incomplete`
- `complete=false`
- `release_ready=false`

## 2. 제품별 복용 조건 약사 재검토

연구 완료 조건은 아니지만 공개 문서 품질을 높이기 위해 권장한다. 2026-07-15 추가한 복용 조건 32개는 허가 용법·용량 PDF의 수치, locator와 SHA-256을 확인하였다. 그러나 2026-07-14 외부 약사 규칙 검토 이후 생성되었으므로 약사 검토 완료로 표시하지 않았다.

약사는 다음 항목을 확인하면 된다.

- 1회 최대 단위와 하루 최대 횟수
- 성분별 최대 1일량
- 최소 복용 간격
- 명시 상한과 허가 용법 범위의 최대값 구분
- 소아 체중별 용법과 제형별 예외

## 3. 선택적 범위 확장

현재 13개 분석 제품은 대표 OTC 후보와 규칙 시나리오 범위다. 국내 판매량 상위 집합으로 주장하지 않는다. 비공개 판매 카탈로그 776건 중 약학정보원 연결이 confirmed인 SKU는 458건이며 고유 공식 품목은 413개다. 약학정보원 자료는 연구용 검색·분류에 연결했지만 식약처 승격 근거가 완료된 카탈로그 후보는 0건이다.

범위를 확장하려면 다음 순서가 필요하다.

1. 약학정보원 `review_required` 10건의 제품명, 제조사, 제형과 경쟁 후보를 사람이 확인한다.
2. `DATA_GO_KR_SERVICE_KEY`를 환경변수로만 제공한다. 키를 파일이나 로그에 저장하지 않는다.
3. 건강기능식품·코스메틱·의료기기·식품을 식약처 제품 영역으로 다시 확인한다.
4. 의약품 후보를 식약처 `item_sequence`, 제품명, 제조사, 제형과 규격으로 확인한다.
5. 일반의약품 구분, 허가 상태, 성분·함량, 용법·용량과 DUR 근거를 연결한다.
6. 식약처 근거와 약학정보원 값이 다르면 양쪽 원문과 충돌을 보존하고 사람이 승격 여부를 결정한다.

판매 SKU 가격은 임상 판정, 판매 순위 또는 현재 판매 상태에 사용하지 않는다.
